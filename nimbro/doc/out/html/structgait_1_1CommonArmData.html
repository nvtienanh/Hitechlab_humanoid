Detailed Description

Data struct that contains the information pertaining to an arm pose that should be common amongst all pose representations.

The assumed joint order is shoulder pitch → shoulder roll → elbow pitch. The upper and lower arm links are assumed to be of the same length (i.e. linkLength).
