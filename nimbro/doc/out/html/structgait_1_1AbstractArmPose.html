Detailed Description

Data struct that encompasses the abstract representation of an arm pose.

The assumed joint order is shoulder pitch → shoulder roll → elbow pitch. The upper and lower arm links are assumed to be of the same length (i.e. cad.linkLength). The correct value of the link length must be set only if setFromInversePose() is used.
