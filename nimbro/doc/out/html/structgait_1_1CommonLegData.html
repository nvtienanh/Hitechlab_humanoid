Detailed Description

Data struct that contains the information pertaining to a leg pose that should be common amongst all pose representations.

The assumed joint order is hip yaw → hip roll → hip pitch → knee pitch → ankle pitch → ankle roll. The upper and lower leg links are assumed to be of the same length (i.e. linkLength).
