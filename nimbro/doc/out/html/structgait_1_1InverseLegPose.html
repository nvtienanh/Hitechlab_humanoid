Detailed Description

Data struct that encompasses the inverse representation of a leg pose.

The assumed joint order is hip yaw → hip roll → hip pitch → knee pitch → ankle pitch → ankle roll. The upper and lower leg links are assumed to be of the same length. The correct value of the link length must be set only if the pose conversion functions are used.
